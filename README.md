🌟 核心功能
自動偵測： 實時同步網頁顯示的「等待時間」，精準鎖定變紅的計時器。

智慧跳轉： 當會員等待達 15 秒且你不在該分頁時，自動將網頁拉回最前面。

防干擾機制： 若你已在客服頁面處理中，機器人會保持安靜，不會強行跳轉干擾你打字。

語音通知： 觸發時會播放「嗶」提示音並跳出系統桌面通知。

🛠️ 安裝步驟 (第一次使用)
下載檔案： 前往 GitHub 連結，點擊綠色的 [Code] 按鈕並選擇 [Download ZIP]。

解壓縮： 將下載的壓縮檔解壓縮到你電腦中一個固定的資料夾（例如：D:\Tools\CustomerBot）。

開啟擴充功能頁面： 在 Chrome 瀏覽器地址欄輸入 chrome://extensions/ 並回車。

開啟開發者模式： 開啟頁面右上角的 「開發者模式」 開關。

載入插件： 點擊左上角的 「載入解壓縮擴充程式」，選取剛才解壓縮的資料夾。

🔄 如何更新 (當開發者更新程式碼時)
由於本插件為手動安裝，當我修改了功能後，請執行以下動作獲取更新：

重新下載： 再次下載 GitHub 上的最新 ZIP 檔。

覆蓋檔案： 將新檔案解壓縮並 直接覆蓋 掉原本資料夾內的舊檔案。

重新整理： 到 Chrome chrome://extensions/ 頁面，找到本插件，點擊右下角的 「🔄 重新整理」 圖示。

⚠️ 注意事項 (必看！)
為了確保機器人正常運作，請檢查以下設定：

啟動音效權限： 每次重新整理網頁後，請在網頁上 隨便點一下滑鼠。瀏覽器規定必須有用戶點擊動作，腳本才能播放提示音。

效能白名單： * 到 Chrome 「設定」 -> 「效能」。

在「記憶體節省模式」中，將 gh.xchatclub.com 加入 「始終保持活動的網站」，防止腳本被瀏覽器凍結。

不要「最小化」： * 請保持客服視窗開啟。你可以用其他視窗（如 Word/Excel）把它 蓋住，但不要點擊右上角的 — 縮小到工作列，否則偵測會變慢。

允許通知： 當瀏覽器詢問是否允許顯示通知時，請選擇 「允許」。

📧 常見問題
Q: 為什麼 15 秒到了沒跳轉？

A: 檢查是否最小化了視窗，或是在 chrome://extensions/ 頁面點擊重新整理。

Q: 為什麼沒有聲音？

A: 刷新網頁後必須先在頁面點擊一下。

Q: 我在打字時它會亂跳嗎？

A: 不會。只要你正看著客服頁面，它就不會執行強制跳轉。
